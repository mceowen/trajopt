# TrajOpt
TrajOpt is a self-contained Python library for multi-segment trajectory optimization using Sequential Convex Programming (SCP).

## Segments

```math
\begin{array}{r}
\text{minimize} & \frac{1}{2}x^T P x + q^T x\\[2ex]
 \text{subject to} & Ax + s = b \\[1ex]
        & s \in \mathcal{K}
 \end{array}
```
